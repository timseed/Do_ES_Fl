Add

To call the Add url you need to pass data in from bash like this.

curl -H "Content-type: text/plain" -X POST http://0.0.0.0:5000/messages --data-binary @file_to_load
To Load multiple files to something like this
find "*/Markdown" | grep md$ | xargs -I {} -t curl -H "Content-type: text/plain" -X POST http://0.0.0.0:5000/add --data-binary @{}

Delete

To reset the data ... then call the /delete api curl 'http://0.0.0.0:5000/delete'
